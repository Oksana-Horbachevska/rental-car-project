RentalCar is a modern car rental application built with Next.js.
The platform provides users with a convenient and intuitive way to browse available vehicles and make rental requests.

The platform allows users to:  
- Browse the full catalog of available cars 
- Filter cars by brand, rental price, and mileage  
- View detailed information about each car
- Submit a booking request by filling out a simple form
